# Roadmap

This document outlines the planned direction for GitSmith. Priorities may shift based on community feedback.

## v0.5 — Stability & Polish

- [ ] Increase test coverage to 80%+
- [ ] Full i18n support with community translations
- [ ] Keyboard-driven workflow (navigable without mouse)
- [ ] Accessibility audit (WCAG 2.1 AA compliance)
- [ ] Performance profiling and optimization for large repos (100k+ commits)

## v0.6 — Collaboration

- [ ] GitHub/GitLab/Bitbucket integration (issues, PRs, CI status)
- [ ] Built-in diff editor for resolving conflicts visually
- [ ] Commit signing (GPG/SSH) UI improvements
- [ ] Multi-repo workspace support

## v0.7 — Advanced Git

- [ ] Interactive rebase drag-and-drop improvements
- [ ] Git hooks management UI
- [ ] Partial clone / sparse checkout support
- [ ] Advanced search across branches and history

## v1.0 — Production Ready

- [ ] Auto-update for all platforms (Windows, macOS, Linux)
- [ ] Plugin/extension system
- [ ] Customizable themes and layouts
- [ ] Comprehensive documentation and tutorials

## Community Contributions

We welcome contributions! Check the [issues](https://github.com/Schengatto/git-expansion/issues) page for `good first issue` and `help wanted` labels.
